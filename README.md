# 日本株ポートフォリオ最適化分析

## 概要
日本株6銘柄を対象に、効率的フロンティアを用いた最適ポートフォリオを構築し、
「過去データによる最適化は未知データでランダムより優れるか」を検証した。

## 分析の問い
「最適化した個別株ポートフォリオは、ランダムな重みより優れるか」

## 使用銘柄
| ティッカー | 銘柄 | セクター |
|------|------|------|
| 7203.T | トヨタ | 自動車 |
| 6758.T | ソニー | テック |
| 8035.T | 東京エレクトロン | 半導体 |
| 8306.T | 三菱UFJ | 金融 |
| 4502.T | 武田薬品 | 医薬品 |
| 9983.T | ファーストリテイリング | 小売 |

## 手法
- 訓練期間：2015〜2019年でシャープレシオ最大化による最適化
- 検証期間：2020〜2026年で最適化ポートフォリオとランダム1000通りを比較
- 日経225との累積リターン比較も実施

## 結果
- 訓練期間の最適重みはソニー・東京エレクトロンに集中（各約44%）
- 検証期間の最適化シャープレシオ：0.903
- ランダム1000通りの中央値：約1.05
- 最適化は検証期間においてランダムの下位に位置した

## 考察
- 訓練期間（2015〜2019）はテック株が好調→最適化がテックに集中
- 検証期間（2020〜2026）は市場環境が大きく変化→集中投資が裏目に出た
- 過去データによる最適化は未知データでランダムより優れるとは言えなかった

## 限界
- 銘柄数が6つと少なくサンプルが限定的
- 検証期間が1つのみであり「たまたま」の可能性を排除できない
- 取引コスト・税金は考慮していない
- 市場環境・期間・銘柄が変われば結論も変わる可能性がある
