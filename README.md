# 日本株ポートフォリオ最適化分析

## 概要
日本株6銘柄を対象に、効率的フロンティアを用いた最適ポートフォリオを構築し、日経225との比較検証を行った。

## 分析の問い
「最適化した個別株ポートフォリオは指数（日経225）に勝てるか」

## 使用銘柄
| ティッカー | 銘柄 | セクター |
|------|------|------|
| 7203.T | トヨタ | 自動車 |
| 6758.T | ソニー | テック |
| 8035.T | 東京エレクトロン | 半導体 |
| 8306.T | 三菱UFJ | 金融 |
| 4502.T | 武田薬品 | 医薬品 |
| 9983.T | ファーストリテイリング | 小売 |

## 手法
- 訓練期間：2015〜2019年でシャープレシオ最大化による最適化
- 検証期間：2020〜2026年で日経225と累積リターンを比較

## 結果
- 最適ポートフォリオは検証期間において日経225を上回るパフォーマンスを示した
- 最適重みはソニー・東京エレクトロンに集中（各約44%）

## 考察・限界
- 訓練期間のテック株上昇トレンドが検証期間に継続したことが主因
- 特定セクターへの集中リスクがあり、トレンド転換時には大きく負ける可能性がある
- 取引コスト・税金は考慮していない
